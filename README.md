# AI Interview Frontend

React + Vite + Tailwind CSS frontend scaffold with React Router, responsive dashboard layout, dark mode, and authentication pages.

## Setup

1. Install Node.js and npm.
2. Run `npm install`.
3. Run `npm run dev`.

## Features

- React Router routing for `Login`, `Register`, `Dashboard`, `Interview`, and `Results`
- Responsive sidebar and top navigation
- Dark mode toggle
- Modern Tailwind dashboard styling
- TypeScript support
